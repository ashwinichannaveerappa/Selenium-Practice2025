# Selenium Practice (Java + TestNG) – Selenium 4 Features

This repository contains practice scripts and hands-on exercises using **Selenium WebDriver 4** with **Java** and **TestNG**. It’s part of my preparation before developing a full-fledged automation framework.

## 🎯 What This Project Covers

- Selenium WebDriver setup with Java and Maven
- Working with modern Selenium 4 features
- Advanced Locators: Relative Locators (`above()`, `below()`, `near()`, etc.)
- Working with new Chrome DevTools Protocol (CDP) APIs for network interception, geolocation, etc.
- Tab/window handling and advanced browser interactions
- Handling waits using `Duration` API and WebDriverManager
- Data-driven testing using TestNG `@DataProvider`
- Basic Page Object Model (POM) structure

## 🧪 Selenium 4 Features Practiced

- Relative Locators (e.g., `withTagName("input").below(...)`)
- `getRect()` for element dimension and location
- CDP Integration: mock geolocation, capture logs, and emulate network
- Improved window and tab management (`newWindow()` API)
- Simplified syntax for waits and actions

## 🛠️ Technologies Used

- Java 11+
- Selenium WebDriver 4.x
- TestNG
- Maven
- WebDriverManager (for driver setup)
- Git + GitHub
- IntelliJ IDEA

## 🚀 Getting Started

### Prerequisites

- Java JDK 11 or later
- Maven installed
- ChromeDriver (or use WebDriverManager)

### How to Run

```bash
mvn clean test
```
